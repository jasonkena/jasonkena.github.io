Jason Ken Adhinarta

I am an incoming PhD student at MIT CSAIL.

Palm oil fresh-fruit-bunch grading (2020)

Black soldier fly instance segmentation (2020)

Built an Arduino-based automated data collection system for larvae, used Mask R-CNN and YOLACT++ for instance segmentation; adapted DCNv2 to be compatible with NVIDIA’s APEX for memory-efficient training. Part of a larger project to utilize BSF larvae for waste management
